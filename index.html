COURSES INCLUDED IN THIS BOOT CAMP

Computer and Technology Basics Course

Average completion time: 40 hours of study time

All of our boot camps begin with this course. It exists as an undercut to learning computer programming and provides students with a firm foundation in technology fundamentals. Students gain the knowledge of how computers work and a well-rounded technology vocabulary. The Computer and Technology Basics Course covers:

  • Clear definitions for every major technology term
  • Algorithm theory and design
  • Basic machine architecture
  • Central processing unit operation
  • Computer network principles
  • Internet design and operation
  • Web browser operation
  • Social media fundamentals
  • Basic security concepts

And more...

Overview of Software Development Course

Average completion time: 32 hours of study time

Due to the importance of the data contained on this course, it is the second course included in all of our boot camps. It covers the basic elements that are fundamental to any computer program, leading to greater comprehension of every computer programming language you will learn in the future. Students will also gain an understanding of the basic actions of a software developer. The Overview of Software Development Course covers:

  • Object-oriented programming (a popular approach to coding) basics
  • Web application (programs that run on the internet) basics
  • Database basics
  • What a software developer actually does
  • How programs are made in this profession
  • The attitude necessary to be successful
  • How to think like a computer programmer
  • Number systems (various ways to count and store amounts)
  • Data structures (methods for organizing information)
  • What flowcharting (diagramming out the basic sequence of a program) is and how it helps you to develop
  • Windows registry (where vital and permanent computer data is stored) basics
  • Command line (a place for entering printed instructions on your computer) basics
  • Writing code with the popular programming language Python

And more...

Sales and Marketing Fundamentals Course

  • Video conferencing basics

Version Control Course

Average completion time: 24 hours of study time

Keeping track of the various stages of a software program as it is created is vital, if only so that if you find that you’re taking the wrong approach on a programming task, you will have a way to get rid of the changes you made and go back to before you tried that approach. Version control allows this as well as other valuable actions related to managing the sometimes lengthy and complex process of making software. The Version Control Course covers:

  • What source control and version control (ways to keep track of various versions of computer programs) are
  • Git (version control software)
  • GitHub (a website and software owned by Microsoft that is used for version control)

And more...

HTML and CSS Course

Average completion time: 88 hours of study time

Students learn how to develop and edit websites on this course. It covers the latest versions of HTML (Hyper Text Markup Language) and CSS (Cascading Style Sheets). All websites are made using HTML. CSS is a tool used to manage many elements of the pages made with HTML. The HTML and CSS Course covers:

  • All the basic fundamentals of HTML5 (the newest version of HTML)
  • All the basic fundamentals of CSS3 (the newest version of CSS)
  • Making an HTML5 website
  • Customizing it with CSS3
  • File Transfer Protocol (how to transmit files [i.e., websites] using the internet)
  • BootStrap (a popular web development framework)

And more...

Version Control Course

Average completion time: 24 hours of study time

Keeping track of the various stages of a software program as it is created is vital, if only so that if you find that you’re taking the wrong approach on a programming task, you will have a way to get rid of the changes you made and go back to before you tried that approach. Version control allows this as well as other valuable actions related to managing the sometimes lengthy and complex process of making software. The Version Control Course covers:

  • What source control and version control (ways to keep track of various versions of computer programs) are
  • Git (version control software)
  • GitHub (a website and software owned by Microsoft that is used for version control)
